Quick Start Guide

Get up and running with the Nmap MCP Server in minutes.

1Make a Request

Send a POST request to the /mcp endpoint:

curl -X POST http://localhost:5001/mcp \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "id": 1,
    "method": "tools/call",
    "params": {
      "name": "nmapScan",
      "arguments": {
        "target": "scanme.nmap.org",
        "flags": "-T4 -F"
      }
    }
  }'

2Get Results

Receive detailed scan results in JSON format:

{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "content": [{
      "type": "text",
      "text": "Nmap Scan Results for scanme.nmap.org\n\nScan started: ...\nHost: 45.33.32.156 (scanme.nmap.org)\nStatus: up\nOpen ports:\n  22/tcp - ssh\n  80/tcp - http\n  ..."
    }]
  }
}

API Reference

nmapScan Tool

Performs network scanning using Nmap with customizable parameters.

POST /mcp
Parameters
  • target (string, required): Domain name, IP address, or CIDR notation
  • flags (string, optional): Nmap scanning flags (default: "-T4 -p 1-1000")
Common Nmap Flags
-T4 Aggressive timing
-p 80,443 Specific ports
-sS SYN scan
-A Aggressive scan
-F Fast scan
--top-ports 100 Top 100 ports

getInfo Tool

Returns information about the Nmap MCP server and its capabilities.

POST /mcp
Parameters

No parameters required.

Deployment Guide

Docker Deployment

# Build the image
docker build -t nmap-mcp-server .

# Run the container
docker run -p 5001:5001 nmap-mcp-server

Render Deployment

  1. Connect your repository to Render
  2. Create a new Web Service
  3. Set the build command: docker build
  4. Set the start command: node server.js
  5. Configure environment variables if needed

Local Development

# Install dependencies
npm install

# Start the server
npm start

# Server will be available at http://localhost:5001
